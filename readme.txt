== Description ==
Completely free and simple plugin to add buy one get one free offers to WooCommerce. No ads, no upsells.

This plugin adds an easy-to-use page to your WordPress admin that allows you to create and customise your own Buy One Get One (BOGO) offers.

Developed by Studio Volte.

Thanks to https://www.svgrepo.com/ for the Cart icon used in our plugin logo.

== Installation ==
We recommend installing the plugin automatically via the WordPress plugin directory.
